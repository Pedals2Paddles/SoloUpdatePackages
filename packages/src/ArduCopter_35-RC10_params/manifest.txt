ArduCopter 3.5-RC10 Parameters
(July 8, 2017 / MCL)

For Pixhawk 2.1 green cube Solos ONLY!

This package contains the required Solo specific ArduCopter parameters only. The Pixhawk firmware and the python files for smart shots are not included in this package. It will only set parameters.

1. Press the Install button below.
2. Parameters will be written to the Solo's Pixhawk.
3. Power cycle the Solo when complete.
